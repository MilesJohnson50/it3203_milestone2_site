Key Concepts

Semantics
The meaning of methods, status codes, and headers.
Idempotency
Repeated requests have the same effect (e.g., PUT/DELETE).
Caching
Reduce latency using validators like ETag and Last‑Modified.
TLS
Encryption/authentication for HTTPS; protects integrity and privacy.
CDN
Distributed network that serves content near users for lower latency.
QUIC
UDP-based transport enabling HTTP/3 with stream-level flow control.
Head‑of‑Line Blocking
TCP issue where one lost packet stalls others; QUIC mitigates it.
